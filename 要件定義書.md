## 1. サービス概要

短縮URLを生成・管理するセルフホスト型Webサービス。  
ユーザーが長いURLを短縮し、短いパス（スラッグ）経由で **301リダイレクト** する。  
個人利用を想定するが、品質・設計には妥協しない。

## 2. 機能要件

### 2.1 URL短縮（コア機能）

| 項目 | 仕様 |
|------|------|
| スラッグ最小長 | **2文字**（例: `example.com/aa`） |
| 自動生成スラッグ | **3文字**（デフォルト）、複数の生成タイプから選択可能 |
| 手動スラッグ | **全Unicode文字列を許可**（絵文字、日本語、記号等） |
| 手動スラッグ最大長 | **128文字** |
| 元URLバリデーション | `http://` または `https://` で始まるURLのみ許可 |
| 重複チェック | 同一スラッグが既に存在する場合はエラーを返す |

#### 自動生成タイプ（Toggle切り替え）

UI上でToggleを使い、以下の生成タイプを切り替えられる:

| タイプ | 使用文字 | 例 | 空間サイズ（3文字） |
|--------|---------|------|--------------------|
| **英数字**（デフォルト） | `a-z A-Z 0-9` | `xK9` | 238,328 |
| **小文字+数字** | `a-z 0-9` | `a7m` | 46,656 |
| **数字のみ** | `0-9` | `482` | 1,000 |
| **発音可能** | 子音+母音の交互パターン | `naho`, `kite` | 可変（4〜6文字） |
| **絵文字** | ランダム絵文字の組み合わせ | `🎸🌊` | 可変（2〜3文字） |

#### スラッグ長さの調整

**Slider** と横の **数値入力Box** で生成するスラッグの文字数を変更できる。  
Sliderを動かすとBoxの値が連動し、Boxに直接入力しても反映される。

| タイプ | 最小長 | 最大長 | デフォルト |
|--------|--------|--------|-----------|
| 英数字 | 2 | 10 | 3 |
| 小文字+数字 | 2 | 10 | 3 |
| 数字のみ | 2 | 10 | 3 |
| 発音可能 | 4 | 12 | 4 |
| 絵文字 | 2 | 6 | 2 |

#### 自動生成のUXフロー

1. 元URLを入力
2. Toggleで生成タイプを選択
3. Slider / 数値Boxで文字数を調整
4. **「生成」ボタン**を押すとスラッグ候補をプレビュー表示
5. 気に入らなければ再度「生成」ボタンで別の候補を生成（何度でも可）
6. 気に入ったスラッグが出たら**「登録」ボタン**で確定・保存

> 「生成」時点ではDBに保存しない。「登録」時に初めてDBへ書き込む。  
> 登録時に衝突が発生した場合はエラーを表示し、再生成を促す。

#### 自動生成の衝突回避

- 生成されたスラッグがDB内に既存の場合、再生成する（最大5回リトライ）
- 英数字タイプの空間が80%以上使用された場合、自動的に4文字に拡張

### 2.2 リダイレクト

| 項目 | 仕様 |
|------|------|
| リダイレクト方式 | **301 Moved Permanently** |
| Unicodeスラッグ | URLエンコード済みパスでもデコード済みパスでもマッチする |
| 存在しないスラッグ | 404ページを返す |

> **注意**: 301はブラウザにキャッシュされるため、リダイレクト先の変更が即座に反映されない場合がある。管理画面での編集時にこの旨を表示する。

### 2.3 管理画面（Compose for Web / Wasm）

#### URL管理

| 機能 | 説明 |
|------|------|
| URL一覧 | 登録済みURL一覧の表示（ページネーション付き） |
| URL作成 | 自動生成（タイプ切り替えToggle＋プレビュー→登録） / 手動スラッグ |
| URL編集 | リダイレクト先URLの変更 |
| URL削除 | 短縮URLの削除 |
| 検索 | スラッグ・元URLでの検索 |

#### アクセス解析ダッシュボード

| 機能 | 説明 |
|------|------|
| クリック数 | 各URLの総クリック数を表示 |
| 時系列グラフ | 日別・時間別アクセス推移 |
| リファラー | 流入元サイト一覧 |
| デバイス情報 | User-Agentからデバイス種別（PC/モバイル/タブレット）を分類 |
| ブラウザ情報 | ブラウザ名・バージョンの集計 |
| 地域情報 | IPアドレスベースの国・地域（GeoIPライブラリ使用） |

### 2.4 認証・ユーザー管理

| 項目 | 仕様 |
|------|------|
| 認証方式 | **パスワード認証**（セッションベース） |
| 複数ユーザー | 対応 |
| ユーザー登録 | **ログイン画面にハッシュ生成ツールを設置** |
| 登録フロー | ハッシュ生成 → 生成されたハッシュ値を直接Xodus DBに投入 |

#### ハッシュ生成ツール（ログイン画面内）

- ログイン画面の下部 or サブページに配置
- ユーザー名とパスワードを入力し、保存用のハッシュ文字列を生成・表示
- 実際の登録はDBへの直接投入（CLI or スクリプト）で行う
- パスワードハッシュには **bcrypt** を使用

### 2.5 API

外部公開APIは **不要**。  
管理画面（Compose Wasm）とKtorサーバー間の内部通信のみ。

## 3. 非機能要件

### 3.1 パフォーマンス

| 項目 | 目標 |
|------|------|
| リダイレクト応答時間 | < 50ms |
| GraalVMネイティブイメージ | 起動時間 < 1秒 |

### 3.2 データ保存

| 項目 | 仕様 |
|------|------|
| データベース | JetBrains **Xodus**（組み込みトランザクショナルDB） |
| バックアップ | Xodusのスナップショット機能を利用 |
| URL有効期限 | **永続**（期限なし。削除は管理画面から手動で行う） |

### 3.3 セキュリティ

- 管理画面へのアクセスは認証必須
- レートリミット: **なし**
- オープンリダイレクト対策: リダイレクト先はHTTP(S)のみ許可
- XSS/CSRF対策
- パスワードは **bcrypt** でハッシュ化して保存

## 5. データモデル

### 5.1 ShortenedUrl

| フィールド | 型 | 説明 |
|-----------|------|------|
| id | String (UUID) | 内部識別子 |
| slug | String | 短縮パス（ユニークキー） |
| originalUrl | String | リダイレクト先URL |
| createdAt | Instant | 作成日時 |
| updatedAt | Instant | 更新日時 |
| clickCount | Long | 総アクセス数 |
| isAutoGenerated | Boolean | 自動生成かどうか |
| createdBy | String | 作成者ユーザーID |

### 5.2 AccessLog（アクセス解析用）

| フィールド | 型 | 説明 |
|-----------|------|------|
| id | String (UUID) | 内部識別子 |
| slugId | String | 対象のShortenedUrl ID |
| accessedAt | Instant | アクセス日時 |
| ipAddress | String | アクセス元IPアドレス |
| userAgent | String | User-Agent文字列 |
| referer | String? | リファラーURL |
| country | String? | 国コード（GeoIP） |
| deviceType | String | PC / Mobile / Tablet |
| browser | String | ブラウザ名 |

### 5.3 User

| フィールド | 型 | 説明 |
|-----------|------|------|
| id | String (UUID) | 内部識別子 |
| username | String | ユーザー名（ユニーク） |
| passwordHash | String | bcryptハッシュ |
| createdAt | Instant | 作成日時 |

## 6. 予約パス

以下のパスは短縮URLのスラッグとして使用不可:

| パス | 用途 |
|------|------|
| `/admin/**` | 管理画面（Compose Wasmの配信） |
| `/internal/**` | 管理画面↔サーバー間の内部API |
| `/health` | ヘルスチェック |
| `/favicon.ico` | ファビコン |
| `/robots.txt` | クローラー制御 |
